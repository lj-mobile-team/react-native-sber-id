# react-native-sber-id

Wrapper native Sber ID SDK for React Native

## Installation

```sh
npm install react-native-sber-id
```

## Usage

```js
import SberId from "react-native-sber-id";

// ...

const result = await SberId.multiply(3, 7);
```
